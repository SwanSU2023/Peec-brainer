A workflow that answers the four questions every SEO team faces with LLMs

Stride Up manages 40+ Peec AI accounts for brands like Lancôme, Kérastase, FnacDarty, Audi, L'Oréal Professionnel. Every Monday, our consultants ask the same questions: which prompts to track, which pages to optimise, why our pages aren't cited and how to reach the domains citing the competitors. Peec Brain answers all four through a plugin of four skills orchestrated over the Peec MCP.

Module 1 — Prompt Discovery Generates Peec prompts by fusing GSC + Ahrefs + client product catalog. Splits branded vs non-branded.
Module 2 — Content Gap Traffic-first prioritisation. Filters non-branded. Enriches with top-3 competitors occupying the gap and page type.
Module 3 — Structural Audit ⭐ Diff editorial structure + schema.org between brand page and LLM-cited pages. Prescriptive content brief.
Module 4 — Citation Authority ⭐ Domains citing competitors but never you, enriched with Ahrefs DR and typology. Outreach-ready CSV.

The full pipeline on Lancôme

The Peec Brain workflow across an agency week

Monday morning. We run Peec Brain on a client account (Lancôme here). Module 1 analyses the existing prompts, cross-references with Ahrefs + GSC, and proposes 22 new prompts across 6 topics (Perfumes, Eye/Face Makeup, Serums/Creams/Diagnostic). After review, we create them in Peec.

Monday afternoon. Module 2 cross-references Peec data with Ahrefs top pages. It surfaces 20 opportunities: high-traffic Lancôme pages barely cited by LLMs. The "routine skincare" page comes first: 1,169 visits/month, 0% visibility on 5 prompts.

Tuesday. Module 3 explains why. On the prompt "What's the best anti-aging serum in 2026 according to dermatologists?", ChatGPT expects 6 sections, one per active ingredient. The target Lancôme page has 0 H2s and 0/6 coverage. Peec Brain emits the AI-ready brief with the 6 recommended H2s and 4 concrete action items.

Wednesday to Friday. The brief feeds the content production. The Cowork artifact (this page) tracks the evolution week over week.

Next Friday. We refresh, measure visibility evolution, prioritise the next batch.

Architecture

Four skills packaged in a Cowork plugin peec-brain, installable in one command by any Peec + Cowork user:

  • peec-prompt-discovery — bootstrap a client account
  • peec-gap-analyzer — weekly opportunity review
  • peec-structural-audit — the per-prompt brief (Module 3)
  • peec-citation-authority — the per-prompt PR outreach CSV (Module 4)
  • peec-brain — orchestrator (onboarding / weekly / adhoc modes)

All consume the Peec MCP server. Python scripts are public on github.com/SwanSU2023/Peec-brainer.
